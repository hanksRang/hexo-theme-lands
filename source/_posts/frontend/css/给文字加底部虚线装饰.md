### 核心代码
```css
text-decoration: underline dotted;
```

### 实例
```html
<span class="ss">这句话展示如何给文字底部加虚线</span>

<style type="text/css">

    .ss {
        color: #062c33;
        text-decoration: underline dotted;
    }

</style>
```
